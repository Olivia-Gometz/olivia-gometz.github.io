Comment récupérer ma signature ?

  1. Cliquez à l'intérieur du cadre blanc.
  2. Cliquez sur CTRL + A pour sélectionner toute la signature (images comprises).
  3. Cliquez sur CTRL + C pour copier le contenu sélectionné.
  4. Rendez-vous dans vos paramètres Gmail pour coller le contenu de votre signature dans l'emplacement prévu à cet effet, puis enregistrez.
